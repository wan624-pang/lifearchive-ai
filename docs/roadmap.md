# LifeArchive AI — 路线图

## MVP (v0.1) — 当前版本

核心目标：验证产品逻辑，完成基础文件处理和 AI 分类流程。

- [x] ZIP 文件上传和解压
- [x] 多格式文件解析（txt, md, pdf, docx, 图片）
- [x] Mock AI 分类器（关键词规则 + 正则日期提取）
- [x] 资料库报告生成
- [x] 自然语言搜索
- [x] Web Dashboard
- [x] 示例数据
- [x] 隐私设计说明

## v0.2 — 真实 AI 接入

- [ ] 接入真实 LLM API（OpenAI-compatible）
- [ ] 图片 OCR（使用 Tesseract 或云 OCR）
- [ ] 重复文件检测（基于内容哈希）
- [ ] 时间线可视化
- [ ] 分类结果人工修正功能
- [ ] 文件预览功能

## v0.3 — 智能搜索与导出

- [ ] 本地向量数据库（ChromaDB / LanceDB）
- [ ] 语义搜索（基于 embedding 相似度）
- [ ] 跨文件关联分析
- [ ] 家庭交接清单 PDF 导出
- [ ] 多语言支持（英文文件识别）
- [ ] 批量文件操作

## v0.4 — 隐私增强

- [ ] 敏感信息自动脱敏
- [ ] 端侧 LLM 集成（Ollama / llama.cpp）
- [ ] 文件加密存储
- [ ] 访问控制和用户隔离
- [ ] 数据导出和清除

## v1.0 — 生产就绪

- [ ] 移动端响应式 / PWA
- [ ] 批量资料库管理
- [ ] 自定义分类规则
- [ ] API 文档完善
- [ ] 性能优化（大文件处理）
- [ ] 多用户支持
- [ ] Docker 一键部署

## 长期愿景

- 家庭共享资料库
- 智能提醒（保险到期、合同续签、体检预约）
- 与主流云存储集成（Google Drive, Dropbox, OneDrive）
- AI 助手对话式资料查询
- 资料完整性检查（缺失文件智能推荐）
