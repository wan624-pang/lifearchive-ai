# LifeArchive AI — 产品文档

## 产品定位

LifeArchive AI 是一个隐私优先的个人数字资料整理工具，帮助用户把散落在各处的文件整理成可搜索、可总结、可交接的私人资料库。

## 目标用户

- 有大量个人文件需要整理的普通用户
- 需要管理家庭资料的家庭管理者
- 准备出国、换工作、买房等需要资料交接的人
- 自由职业者和小微企业主的资料管理
- 帮助老人整理和交接家庭资料

## 核心场景

用户上传一个 ZIP 文件夹，里面包含 PDF、图片、txt、markdown、docx 等资料。系统自动完成：

1. 解压并读取文件列表
2. 解析每个文件的内容
3. 使用 AI 进行分类、摘要、标签提取
4. 识别涉及的人物、机构、重要日期
5. 判断敏感等级
6. 推荐归档目录
7. 生成个人资料总览报告
8. 提供可搜索的文件索引

## AI 工作流（5 步）

### Step 1: Ingest（摄入）
- 读取 ZIP 文件，解析文件结构
- 遍历目录，提取每个文件的基础 metadata（文件名、大小、扩展名）
- 过滤支持的文件类型

### Step 2: Parse（解析）
- 文本文件（.txt, .md）：直接读取 UTF-8 内容
- PDF：使用 pypdf 提取文字
- DOCX：使用 python-docx 提取段落文本
- 图片（.jpg, .png）：保留路径和 metadata，标记为「需要 OCR」
- 每个文件生成统一的 DocumentRecord

### Step 3: Classify（分类）
- 使用 AI 或 Mock 分类器对每个文件进行分析
- 输出：类别、摘要、标签、人物、机构、日期、敏感等级、推荐目录、置信度
- Mock 模式使用关键词匹配规则
- LLM 模式调用 OpenAI-compatible API

### Step 4: Aggregate（聚合）
- 汇总所有文件的分类结果
- 生成类别统计
- 提取所有重要日期并排序
- 识别重复或疑似重复文件
- 标记缺失的常见资料类别

### Step 5: Report（报告）
- 生成完整的资料库总览报告
- 包含：概述、分类统计、时间线、缺失提醒、风险提示、交接清单、建议
- 报告存储在数据库中，可随时查看

## 文件类别

| 类别 | 敏感等级 | 说明 |
|------|----------|------|
| 身份与证件 | high | 身份证、护照、户口本、驾照等 |
| 合同与法律 | high | 租房合同、劳动合同、协议等 |
| 医疗与健康 | high | 体检报告、病历、处方等 |
| 保险与资产 | high | 保险保单、理财合同等 |
| 房屋与车辆 | medium | 房产证、车辆登记、维修记录等 |
| 学习与证书 | medium | 毕业证、学位证、培训证书等 |
| 工作资料 | medium | 工作报告、项目文档等 |
| 发票与报销 | medium | 发票、报销单、费用记录等 |
| 家庭纪念 | low | 家庭照片、通讯录等 |
| 旅行与照片 | low | 旅行计划、行程单等 |
| 待确认 | low | 无法明确分类的文件 |

## 搜索功能

支持自然语言搜索，系统会解析用户意图：

- 类别过滤：「找出所有保险相关文件」
- 存在性检查：「我有没有上传身份证」
- 时间过滤：「列出所有2024年的医疗资料」
- 状态过滤：「哪些文件需要人工确认」
- 敏感过滤：「哪些文件敏感等级最高」

MVP 使用关键词搜索 + metadata 过滤，代码结构预留向量数据库接口。

## 资料库报告

报告包含：
- 文件总数和类别分布
- 高敏感文件提醒
- 重要日期时间线
- 缺失材料提醒（常见类别中没有的）
- 重复/疑似重复文件提醒
- 需要人工确认的文件列表
- 家庭交接清单草稿
- 下一步行动建议
